🏛️ AI Governance Decision Dashboard

An intelligent data-driven governance system that analyzes public complaints using NLP-based severity scoring and provides actionable policy recommendations.

🚀 Features
📊 Real-time complaint analytics dashboard
🔍 Department & location filtering
🌡️ Severity heatmap visualization
🤖 AI-based decision engine
🧠 NLP-powered complaint severity scoring
📈 Interactive charts (Plotly)
🧠 How It Works

The system uses:

Weighted keyword NLP
Rule-based inference
Data aggregation techniques

To identify:

Critical departments
High-risk locations
Policy intervention priorities
